# 주요기능

1. 사용자가 질문을 업로드 할 수 있는 기능
2. 사용자가 다른 사용자들의 질문들에 투표할 수 있는 기능
3. 게시글 검색 기능
4. 게시글에 댓글을 달 수 있는 기능
5. 해당 게시글을 공유할 수 있는 기능
